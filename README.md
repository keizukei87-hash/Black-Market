- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- Google Fonts (JetBrains Mono)

## 📦 Cara Deploy ke GitHub Pages
1. Fork repository ini
2. Buka Settings → Pages
3. Pilih branch `main` dan folder `/ (root)`
4. Klik Save
5. Website akan live di `https://username.github.io/blackmarket`
